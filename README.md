# Amy's Character Generator

A simple, free character generator for students and writers.

## Files

- `index.html` — the complete tool

## How to publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`.
3. Go to **Settings**.
4. Click **Pages**.
5. Under **Source**, choose **Deploy from a branch**.
6. Branch: `main`
7. Folder: `/ (root)`
8. Click **Save**.

Your live link will look like:

`https://yourusername.github.io/your-repository-name/`
